# Examples

This directory contains example projects demonstrating claude-dotfiles usage.

## Available Examples

### [demo-project/](./demo-project/)

A minimal Todo API example showing:

- How to structure a CLAUDE.md file
- Project-specific settings in `.claude/settings.json`
- How Claude uses the configuration

**Try it:**
```bash
cd examples/demo-project
claude
# Ask: "Explain this project to me"
```

## Creating Your Own Examples

Want to contribute an example? Great! Each example should include:

1. `CLAUDE.md` - Demonstrating the configuration
2. `README.md` - Explaining what the example shows
3. `.claude/settings.json` - (Optional) Project-specific settings

See [CONTRIBUTING.md](../CONTRIBUTING.md) for submission guidelines.
